Root element

.p-code-snippet

Code snippet in default variant.

&.is-bordered

Bordered version of the code snippet. To be used when some borderless elements such as iframe are used inside the code snippet.

Code block

.p-code-snippet__block

Notification in default variant. Same as .p-notification--information

.p-code-snippet__block--numbered

Information notification. Same as default .p-notification.

.p-code-snippet__block--icon

Information notification. Same as default .p-notification.

&.is-windows-prompt

Information notification. Same as default .p-notification.

&.is-url

Information notification. Same as default .p-notification.

&.is-wrapped

Information notification. Same as default .p-notification.

Code line

.p-code-snippet__line

Single line of code block. This is needed in the numbered style with `.p-code-snippet__block--numbered`.

Header

.p-code-snippet__header

Code snippet header in default style.

&.is-stacked

Stacked version of the header.

Title

.p-code-snippet__title

The title of the code block. This is part of header element.

Dropdowns container

.p-code-snippet__dropdowns

The container element for code snippet dropdowns in the header.

Dropdown

.p-code-snippet__dropdown

Single dropdown inside the header.
